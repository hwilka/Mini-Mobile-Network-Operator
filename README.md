# Mini Mobile Network Operator

This is a small game I did in under 48 hours for the Ludum Dare 59 game jam.


## Code Quality
The code of the project was written in under 48 hours and is mostly a mess without 
comments. Nevertheless I wanted to publish the original version I used in the game jam.

## Engine
The game was written using the Godot Engine (Version 4.6.1),
[which is licensed under the MIT License](https://github.com/godotengine/godot/blob/master/LICENSE.txt).
